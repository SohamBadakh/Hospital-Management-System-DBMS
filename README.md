1. ## Project Title
 Hospital Management System (DBMS Project)

2. ## Short Description of Project
This project is a Hospital Management System developed using MySQL. 
It manages patient records, doctor details, consultations, billing, and test reports using relational database concepts.

3. ## Features
- Manage patient information
- Manage doctor records
- Track consultations (Many-to-Many relationship)
- Billing management
- Test report management

4. ## Database Concepts Used in project
- ER Diagram Design
- Primary & Foreign Keys
- JOIN Queries
- Aggregate Functions
- Subqueries
- Constraints & Referential Integrity

5. ## Project Structure
- schema.sql - Contains table creation queries
- data.sql - Contains data insertion queries
- queries.sql - Contains important SQL queries
- ERD.png - Entity Relationship Diagram
- screenshot Folder - Contains screenshot of Tables and Result tables of Join and Subquery.

6. ## How to Run the Project
- 1. Open MySQL command line
     (mysql -u root -p)
- 2. Create a new database:
     CREATE DATABASE hospital_management;
- 3. Use the database:
     USE hospital_management;
- 4. Run schema.sql
- 5. Run data.sql
- 6. Execute queries from queries.sql
